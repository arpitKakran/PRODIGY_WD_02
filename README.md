This is a simple Stopwatch web application built using HTML, CSS, and JavaScript.

It was made as part of Task-02 during my internship at Prodigy Infotech.

Features

- Start the stopwatch
- Pause and resume
- Reset to 00:00:00
- Record lap times
- Clean and modern dark gradient UI

Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla)
